# Helpdesk Tickets System

This is a web-based helpdesk application built to manage and track support tickets efficiently.
The goal of this project is to help users submit support requests, monitor ticket status, 
and allow staff to manage and resolve issues in an organized way.

## What the application does

- User registration and login
- Submit and manage support tickets
- Track ticket status (open and closed)
- Multi-page navigation interface
- Secure session handling
- Database-driven ticket storage

## Technologies

- PHP
- MySQL
- HTML and CSS
- JavaScript
- XAMPP (local server)

## How to run the project

1. Clone the repository:
[https://github.com/LeinadaMisku/helpdesk-tickets-system.git](https://github.com/LeinadaMisku/helpdesk-tickets-system.git)
2. Move the folder into `xampp/htdocs`
3. Create a database named: `helpdesk_db`
4. Import the SQL file if included
5. Start Apache and MySQL in XAMPP
6. Open in browser: [http://localhost/helpdesk-tickets-system](http://localhost/helpdesk-tickets-system)

## About this project

This project is a full-stack web-based helpdesk system developed using PHP 
and MySQL for the backend, and HTML, CSS and JavaScript for the frontend. 
It provides a complete ticket management workflow with user authentication, 
session handling, and database-driven storage. The goal was to build a 
real-world application that integrates frontend design with backend logic, 
developed collaboratively as a university group project.
